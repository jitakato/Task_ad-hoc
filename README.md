# Методология
Тестирование проводилось в формате ad hoc для главной страницы сайта-питомника "Собаседник". В процессе работы использовалась баг-трекинговая система Jira.
Для каждого дефекта был заведен свой баг-репорт, описаны шаги по воспроизведению, ожидаемый и фактический результаты. Также каждый баг-репорт подкреплен скриншотом, который вкратце показывает расположение бага на странице пользователя и ожидаемый результат.
